## 学习路线

```
第一阶段：理解基础 (30分钟)
  └─ 阅读 01-基础入门.md
       └─ 了解插件是什么、能做什么、核心理念

第二阶段：熟悉工具 (1小时)
  └─ 阅读 02-核心Skills详解.md
       └─ 逐个了解每个 Skill 的功能和触发条件

第三阶段：动手实践 (2小时)
  └─ 阅读 03-开发工作流实战.md
       └─ 跟着示例走一遍完整流程

第四阶段：深入精通 (持续)
  └─ 阅读 04-进阶技巧与最佳实践.md
       └─ 掌握高级用法和优化策略

日常使用：
  └─ 随时查阅 05-快速参考卡.md
```
